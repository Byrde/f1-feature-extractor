# F1 Session Feature Extractor

Extract meaningful KPIs from F1 free practice sessions (via [OpenF1 API](https://openf1.org)) for GridRival fantasy team selection.

## Setup

```bash
npm install
```

## Usage

```bash
npm run build        # Compile TypeScript
npm run typecheck    # Type-check without emitting
npm test             # Run tests
npm run test:watch   # Run tests in watch mode
```
